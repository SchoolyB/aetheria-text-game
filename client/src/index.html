Welcome To Aetheria

A terminal based text adventure/fantasy/rpg game written in "C" that has quickly become the passion project of its sole developer Marshall Burns. From this site you can access media files from the games development as well as patch notes from the developer.


Clone, Compile, & Play!

Follow these simple steps to begin your journey in Aetheria

**This program assumes you have a GCC installed on your system**


Clone the repository

git clone https://github.com/SchoolyB/aetheria-text-game.git
Repository Cloned!

Navigate to the "scripts" directory

cd your/file/path/aetheria-text-game/scripts

OPTIONAL: To compile WITHOUT running the program use "./compile.sh"

./compile.sh
Build Successful!

To compile AND run the program use "./execute.sh"

./execute.sh
Build Successful!
Clearing Terminal
Running Aetheria...
